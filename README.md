# Projeto-Pontos-de-Alagamentos-
🎯 Objetivos do Projeto COLAI
O Projeto COLAI tem como objetivo capacitar o grupo a identificar, analisar e propor soluções inovadoras para enchentes e pontos de alagamento, utilizando tecnologia, inteligência artificial e pensamento computacional. Por meio de simulações no Minecraft Education, programação com MakeCode e apoio de ferramentas como GitHub e GitHub Copilot, o grupo desenvolve protótipos (MVPs) capazes de monitorar riscos, gerar alertas e apoiar a prevenção de impactos sociais, promovendo cidades mais seguras, resilientes e sustentáveis.
📌 Pontos Importantes do Projeto COLAI
🧠 Aprendizados Desenvolvidos

Compreensão de problemas reais de impacto social, como enchentes e pontos de alagamento
Desenvolvimento do pensamento computacional (analisar, decompor e resolver problemas)
Aprendizado de lógica de programação (if, else, loops, variáveis)
Trabalho em equipe, colaboração e organização de projetos por etapas (sprints)
Capacidade de transformar uma ideia em um MVP funcional


🛠️ Utilização das Ferramentas

Minecraft Education para simular cenários urbanos e pontos de alagamento
MakeCode para criar códigos que monitoram nível da água e acionam alertas
Git e GitHub para versionamento, histórico do projeto e trabalho colaborativo
Uso de Issues, Backlog e Branches para organizar as tarefas do grupo
Documentação do projeto por meio de README.md


🤖 Inteligência Artificial como Facilitadora

Utilização da IA (GitHub Copilot) como apoio na:

criação de códigos iniciais
sugestão de melhorias na lógica
correção de erros
refatoração do código


A IA atua como uma assistente de aprendizado, explicando trechos de código
Acelera o desenvolvimento do MVP, permitindo que o grupo foque mais na solução do problema
Estimula a autonomia criativa, sem substituir o raciocínio humano
Ajuda a transformar ideias em código de forma mais rápida e segura


🌍 Impacto Social Gerado

Conscientização sobre os riscos causados por enchentes
Proposição de soluções digitais para prevenção e alerta
Uso da tecnologia como ferramenta para melhorar a qualidade de vida nas cidades
Formação de jovens mais preparados para resolver desafios reais com tecnologia e IA
